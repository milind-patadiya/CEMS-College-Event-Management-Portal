# CEMS - College Event Management System

## Description
This project is a web-based application designed to manage college events efficiently. It allows admin, faculty, and students to interact on a single platform.

## Features
- User Authentication
- Event Creation & Management
- Student Registration
- Attendance Tracking
- Notifications

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Install XAMPP
2. Place project in htdocs
3. Import cems_db_final.sql in phpMyAdmin
4. Run localhost
